# 🧠 Diabetic Retinopathy Detection using AI

## 🚀 Project Overview

Diabetic Retinopathy is one of the leading causes of blindness. Early detection can help prevent severe vision loss.
This project uses Deep Learning to analyze retinal images and classify different stages of Diabetic Retinopathy.

---

## 💡 Features

* 📸 Image-based disease detection
* ⚡ Fast and automated predictions
* 🧠 Deep Learning model (CNN)
* 🌐 Web App using Streamlit
* 👨‍⚕️ Helps in early diagnosis

---

## 🛠️ Tech Stack

* Python
* TensorFlow / Keras
* OpenCV
* Streamlit

## ▶️ How to Run

### 1️⃣ Clone Repository

git clone https://github.com/pratiksingh519/diabetic-retinopathy-ai.git
cd diabetic-retinopathy-ai

### 2️⃣ Install Dependencies

pip install tensorflow opencv-python streamlit numpy pillow

### 3️⃣ Run Web App

streamlit run app.py
